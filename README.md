# Sistem Kasir Angkringan (C++)

Program sederhana berbasis **C++** yang mensimulasikan sistem kasir pada sebuah angkringan. Aplikasi ini memungkinkan pengguna memilih menu makanan dan minuman, menghitung total harga berdasarkan jumlah pesanan, memberikan diskon secara otomatis sesuai total belanja, menerima pembayaran dari pelanggan, serta menghitung uang kembalian.

## Fitur

* Menampilkan daftar menu makanan dan minuman.
* Pemesanan dengan jumlah porsi.
* Perhitungan total harga secara otomatis.
* Diskon berdasarkan total pembelian:

  * 6% untuk belanja ≥ Rp25.000
  * 8% untuk belanja ≥ Rp50.000
* Input pembayaran pelanggan.
* Perhitungan uang kembalian atau kekurangan pembayaran.
* Tampilan sederhana menggunakan terminal (console).

Proyek ini dibuat sebagai latihan dasar pemrograman C++ dengan menerapkan konsep **fungsi (function)**, **variabel global**, **percabangan (if/switch)**, **perulangan (do-while)**, **array**, dan **input/output** menggunakan `iostream`.
